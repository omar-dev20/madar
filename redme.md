# MaDar E-Commerce Platform & Dashboard

MaDar is a fully interactive E-Commerce platform and management dashboard built to manage products and view statistics, powered by dynamic data integration.

منصة مدار هي عبارة عن متجر إلكتروني متكامل ولوحة تحكم تفاعلية لإدارة المنتجات وعرض الإحصائيات، تعتمد بالكامل على جلب البيانات الديناميكية وتحديثها.

---

## Core Features | المميزات الرئيسية

* Interactive Dashboard | لوحة تحكم تفاعلية
* Dynamic Product Cards | بطاقات منتجات ديناميكية
* Category Filtering | تصفية حسب الأقسام
* Instant Search | محرك بحث فوري
* Dynamic Shopping Cart | سلة مشتريات ديناميكية
* Responsive Design | تصميم متجاوب

---

## Tech Stack | التقنيات المستخدمة

* HTML5 & CSS3
* JavaScript (Vanilla JS)
* DummyJSON API
* Git & GitHub
